# Name Generator

A simple web application that generates random **Zambian first names** and **Japanese last names** to create unique, culturally blended names.

This project demonstrates random data generation using a full-stack setup with EJS templates and Express.

---

## ✨ Features

- 🎲 Random Zambian first name generation
- 🗾 Random Japanese last name generation
- 🔄 One-click name generation
- ⚡ Fast and lightweight
- 💡 Clean and simple UI
- 🖥️ Server-side rendering using EJS

---

## 🛠 Built With

- **Node.js**
- **Express.js**
- **EJS**
- **CSS**
- **JavaScript**

---

## 🧠 How It Works

The application stores:

- A list of Zambian first names
- A list of Japanese surnames

When the **Generate Name** button is clicked:

1. A random Zambian first name is selected on the server.
2. A random Japanese last name is selected.
3. The names are combined.
4. The result is rendered dynamically using EJS.
5. The generated name is displayed in the browser.
